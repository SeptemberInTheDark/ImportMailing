# ImportMailing

Тестовый Django‑проект для **импорта рассылок из XLSX** с защитой от повторной обработки по `external_id` и имитацией отправки писем через логирование с задержкой.

## Требования

- Python **3.10+**
- Django **4.2+**

## Быстрый старт (локально)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

python manage.py migrate
python manage.py import_mailings ./data/mailings.xlsx
```

## Формат XLSX

Первая строка — заголовки колонок:

- `external_id` — уникальный идентификатор записи во внешней системе (используется для дедупликации)
- `user_id` — идентификатор пользователя
- `email` — email получателя
- `subject` — тема письма
- `message` — текст письма

## Что делает команда

- читает XLSX в потоковом режиме (подходит для больших файлов);
- валидирует строки;
- создаёт записи в БД и **не обрабатывает повторно** уже импортированные `external_id`;
- “отправляет письма” как **запись в лог** с небольшой задержкой;
- по завершении выводит статистику:
  - количество обработанных строк;
  - количество созданных записей;
  - количество пропущенных записей;
  - количество ошибочных строк.
